---
title: "Add dns_lookup operator"
type: feature
---

The new `dns_lookup` operator performs DNS lookups to resolve IP addresses to
hostnames (reverse DNS) or hostnames to IP addresses (forward DNS).

```tql
# Reverse DNS lookup
from {ip: 8.8.8.8}
dns_lookup ip
# {ip: 8.8.8.8, dns_lookup: {hostname: "dns.google"}}

# Forward DNS lookup with custom result field
from {domain: "example.com"}
dns_lookup domain, result=ip_info
# {domain: "example.com", ip_info: {records: [{address: 93.184.215.14, type: "A", ttl: 5m}]}}
```
