Computes an XXH3 hash digest.

```tql
hash_xxh3(x:any, [seed=string]) -> string
```

## Description

The `hash_xxh3` function calculates a 64-bit XXH3 hash digest for the given
value `x`.

## Examples

### Compute an XXH3 digest of a string

```tql
from {x: hash_xxh3("foo")}
```

```tql
{x: "ab6e5f64077e7d8a"}
```
