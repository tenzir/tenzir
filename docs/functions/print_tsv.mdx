Prints a record as a tab-separated string of values.

```tql
print_tsv(input:record, [list_separator=str, null_value=str]) -> string
```

import Docs from '../../.../../../../partials/functions/PrintCSV.mdx';

<Docs operator="print_tsv" separator_text="tab" list_sep="," null_value="-"/>

## Examples

```tql
from {
  x:1,
  y:true,
  z: "String"
}
output = this.print_tsv()
```

```tql
{
  x: 1,
  y: true,
  z: "String",
  output: "1\ttrue\tString",
}
```
