Saves a byte stream via FTP.

```tql
save_ftp url:str [tls=bool, skip_peer_verification=bool, cacert=string,
                  certfile=string, keyfile=string,
                  tls_minium_version=string, tls_ciphers=string]
```

## Description

Saves a byte stream via FTP.

### `url: str`

The URL to request from. The `ftp://` scheme can be omitted.
