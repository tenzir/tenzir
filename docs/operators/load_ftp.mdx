Loads a byte stream via FTP.

```tql
load_ftp url:str [tls=bool, cacert=string, certifle=string,
                  keyfile=string, skip_peer_verification=bool]
```

## Description

Loads a byte stream via FTP.

### `url: str`

The URL to request from. The `ftp://` scheme can be omitted.
