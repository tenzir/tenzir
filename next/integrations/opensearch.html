OpenSearch

OpenSearch is a search and observability suite for unstructured data. Tenzir can send events to OpenSearch.

When sending data to OpenSearch, Tenzir uses the Bulk API and attempts to maximally batch events for throughput, accumulating multiple events before shipping them within a single API call. You can control batching behavior with the max_content_length and send_timeout options.

For more details, see the documentation for the to_opensearch operator.

Tenzir can also present an OpenSearch-compatible REST API via the from_opensearch operator.
