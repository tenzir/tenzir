Quickstart

Get up and running quickly with this tutorial. It just takes a few steps to run pipelines.

Explore the demo node​

The easiest way to get started with Tenzir is using our cloud demo setup:

  1. Create a free account by signing in:

  1. You will be redirected to the Pipelines page:

  1. Create a demo node by clicking the Add node button in the left pane and select Cloud-hosted demo node. The demo node takes 1-2 minutes to boot up:

  1. The demo node comes with the Demo Node Pipelines package from the library pre-installed. Head over to the Library page to view the installed package:

  1. Let's run one of the snippets included with the demo package. This brings you right into the Explorer, where you can interactively run pipelines:

  1. Start learning TQL and explore the data set visually, e.g., by running aggregations and plotting data:

Up next​

Now that you got a first impression of Tenzir, onboard your own node and dive deeper by

  • following the user guides with step-by-step tutorials of common use cases,
  • learning more about the language, operators, connectors, formats, the data model, and
  • understanding why we built Tenzir and how it compares to other systems.

Don't forget that we're here to help! If you have any questions, swing by our friendly community Discord or open a GitHub discussion.
