# print_csv

Prints a record has a comma separated string of values.

```tql
print_csv(input:record, [list_separator=str, null_value=str]) -> string
```

<Docs operator="print_csv" separator_text="comma" list_sep=";" null_value=""/>

## Examples

```tql
from {x:1, y:true, z: "String"}
output = this.print_csv()
```
```tql
{
  x: 1,
  y: true,
  z: "String",
  output: "1,true,String",
}
```
