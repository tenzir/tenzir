# print_ssv

Prints a record has a space separated string of values.

```tql
print_ssv(input:record, [list_separator=str, null_value=str]) -> string
```

<Docs operator="print_ssv" separator_text="space" list_sep="," null_value="-"/>

## Examples

```tql
from {x:1, y:true, z: "String"}
output = this.print_ssv()
```
```tql
{
  x: 1,
  y: true,
  z: "String",
  output: "1 true String",
}
```
